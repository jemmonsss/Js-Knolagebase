# My Knowledge Base

A customizable, wiki-style knowledge base website hosted on GitHub Pages using Jekyll.

## Features

- Clone and deploy to GitHub Pages in minutes
- Add new pages by dropping Markdown files into `_wiki/<category>/`
- Modular configuration via `_config.yml` and data files
- Responsive design with hamburger menu for mobile and tablet
- Client-side search
- Self-documenting manual at `/wiki/using-this-wiki/`

## Quick Start

1. Use this template or clone the repository.
2. Update `_config.yml` with your site title, URL, and baseurl.
3. Edit `_data/navigation.yml` and `_data/categories.yml`.
4. Enable GitHub Pages (Source: GitHub Actions).
5. Push to `main` and watch it build.
