# Getting Started

Welcome! This section will take you from zero to a fully functional knowledge base. You don't need to be a developer to use this site, but some basic familiarity with GitHub and Markdown will help.

## What you'll learn

1. [Quick Start](/wiki/getting-started/quick-start/) — get a copy of this site live on GitHub Pages in under five minutes.
2. [Installation](/wiki/getting-started/installation/) — set up a local copy for editing, previewing, and customization.

## Prerequisites

Before you begin, make sure you have:

- A **GitHub account** (free)
- **Git** installed locally, or GitHub Desktop
- A text editor (VS Code, Notepad++, or any plain-text editor)
- Basic comfort with **Markdown** — see [Adding Content](/wiki/using-this-wiki/adding-content/) if you need a refresher

## Still stuck?

If you run into problems, check [Managing Repo](/wiki/using-this-wiki/managing-repo/) for troubleshooting tips, or open an issue in the repository.
